# Конвертер валют - Python версия

CLI-утилита для конвертации валют с загрузкой актуальных курсов через API.

## Требования

- Python 3.7 или выше
- pip (менеджер пакетов Python)
- Интернет-соединение для получения курсов валют

## Установка

### Шаг 1: Установка зависимостей

```bash
cd Python
pip install -r requirements.txt
```

Будут установлены следующие библиотеки:
- **requests** - для HTTP запросов к API
- **colorama** - для кроссплатформенного цветного вывода

### Шаг 2: Проверка установки

```bash
python --version
```

Убедитесь, что версия Python 3.7 или выше.

## Использование

### Основной синтаксис

**Интерактивный режим:**
```bash
python main.py
```

**Режим с аргументами:**
```bash
python main.py [ИСХОДНАЯ_ВАЛЮТА] [ЦЕЛЕВАЯ_ВАЛЮТА] [СУММА]
```

**История конвертаций:**
```bash
python main.py --history
```

**JSON вывод:**
```bash
python main.py --json USD RUB 100
```

Где:
- `ИСХОДНАЯ_ВАЛЮТА` - код валюты, из которой конвертируем (USD, EUR, RUB и т.д.)
- `ЦЕЛЕВАЯ_ВАЛЮТА` - код валюты, в которую конвертируем
- `СУММА` - количество для конвертации (число)

**Примечание:** Коды валют можно вводить как **большими**, так и **маленькими** буквами (USD, usd, Usd - все варианты работают).

## Примеры использования

### Базовые примеры:

```bash
# Конвертация 100 долларов в рубли
python main.py USD RUB 100

# Конвертация 50 евро в доллары (маленькими буквами)
python main.py eur usd 50

# Конвертация 1000 рублей в евро
python main.py RUB EUR 1000

# Конвертация 200 фунтов в японские иены (смешанный регистр)
python main.py Gbp jpy 200
```

### Различные валюты:

```bash
# Швейцарский франк в канадский доллар
python main.py CHF CAD 150

# Китайский юань в индийскую рупию
python main.py CNY INR 500

# Австралийский доллар в британский фунт
python main.py AUD GBP 300
```

### Интерактивный режим:


## Особенности реализации

- **requests** - надежная библиотека для HTTP запросов
- **colorama** - кроссплатформенная поддержка цветного вывода (работает на Windows, Linux, macOS)
- Автоматическая настройка кодировки UTF-8 для Windows
- Автоматическое приведение кодов валют к верхнему регистру
- Читаемый и понятный код на Python
- Подробная обработка ошибок с понятными сообщениями
- Отображение времени, прошедшего с последнего обновления курсов
- Поддержка кириллицы и Unicode символов

## API

Проект использует бесплатный API от [exchangerate-api.com](https://www.exchangerate-api.com/):
- Не требует регистрации
- Обновление курсов несколько раз в день
- Надежный и быстрый сервис

## Обработка ошибок

Программа корректно обрабатывает следующие ошибки:

### Ошибки ввода:
- Неверный формат суммы (не число)
- Отрицательная или нулевая сумма
- Несуществующая валюта

### Сетевые ошибки:
- Отсутствие интернет-соединения
- Таймаут запроса (10 секунд)
- Ошибки API

### Системные ошибки:
- Проблемы с кодировкой (автоматически решаются для Windows)

Все ошибки выводятся с понятными сообщениями на русском языке.

## Структура кода

```
Python/
├── requirements.txt  # Зависимости проекта
├── main.py           # Основной код программы
└── README.md         # Этот файл
```

### Основные функции:

- `main()` - точка входа в программу
- `get_exchange_rates()` - получение курсов валют из API
- `convert_currency()` - конвертация валюты
- `print_result()` - форматированный вывод результата
- `format_time_ago()` - форматирование времени с последнего обновления
- `get_input()` - получение и валидация ввода от пользователя
- `get_amount()` - получение и валидация суммы для конвертации
- `save_to_history()` - сохранение конвертации в историю
- `show_history()` - отображение истории конвертаций
- `output_json()` - вывод результата в JSON формате
- `output_csv()` - вывод результата в CSV формате
- `output_error()` - вывод ошибок в JSON/CSV формате
- `load_config()` - загрузка конфигурации из config.json
- `print_table()` - вывод результатов в виде таблицы
- `load_cache()` / `save_cache()` - чтение и запись кэша курсов

## Новые возможности

### История конвертаций

Каждая конвертация автоматически сохраняется в файл `history.json` в директории программы. Просмотр истории:

```bash
python main.py --history
```

Формат записи истории:
```json
[
  {
    "timestamp": "2026-01-02T15:30:45.123456",
    "from_currency": "USD",
    "to_currency": "RUB",
    "amount": 100.0,
    "result": 9150.50,
    "exchange_rate": 91.505,
    "rate_update_time": "2026-01-02T12:00:00"
  }
]
```

### JSON вывод

Для использования в скриптах и автоматизации используйте флаг `--json`:

```bash
python main.py --json USD RUB 100
```

Пример ответа:
```json
{
  "success": true,
  "timestamp": "2026-01-02T15:30:45.123456",
  "from_currency": "USD",
  "to_currency": "RUB",
  "amount": 100.0,
  "result": 9150.50,
  "exchange_rate": 91.505,
  "rate_update_time": "2026-01-02T12:00:00"
}
```

При ошибке:
```json
{
  "success": false,
  "error": "описание ошибки"
}
```

### CSV вывод

Для экспорта данных используйте флаг `--csv`:

```bash
python main.py --csv USD RUB 100
```

Пример вывода:
```
2026-01-25T19:06:30.126197,USD,EUR,50.00,42.45,0.849000
```

Формат: `timestamp,from,to,amount,result,rate`

### Множественная конвертация

Передайте несколько целевых валют через запятую — один запрос к API:

```bash
python main.py USD RUB,EUR,CNY 100
```

### Табличный режим

Флаг `--table` выводит результаты в виде отформатированной таблицы — удобно при конвертации в несколько валют:

```bash
python main.py --table USD RUB,EUR,CNY,GBP,JPY 100
```

Пример вывода:

```
  Конвертация 100.00 USD
  ┌──────────┬────────────────┬──────────────┐
  │ Валюта   │ Результат      │ Курс         │
  ├──────────┼────────────────┼──────────────┤
  │ RUB      │ 7759.00        │ 77.5900      │
  │ EUR      │ 86.10          │ 0.8610       │
  │ CNY      │ 623.50         │ 6.2350       │
  │ GBP      │ 73.20          │ 0.7320       │
  │ JPY      │ 11450.00       │ 114.5000     │
  └──────────┴────────────────┴──────────────┘

  Последнее обновление: 2026-03-04 03:00:00 (5 часов назад)
```

### Конфигурационный файл

Создайте `config.json` в директории программы (или скопируйте `config.json.example` из корня проекта):

```json
{
  "default_from": "USD",
  "default_to": "RUB",
  "output_format": "text"
}
```

**Параметры:**
- `default_from` — валюта по умолчанию (подставляется в интерактивном режиме, Enter для подтверждения)
- `default_to` — целевая валюта по умолчанию
- `output_format` — формат вывода: `"text"`, `"json"` или `"csv"` (перебивается флагами `--json`/`--csv`)

## Возможные проблемы и решения

### Проблема: ModuleNotFoundError: No module named 'requests'
**Решение:** Установите зависимости:
```bash
pip install -r requirements.txt
```

### Проблема: Символы не отображаются корректно в Windows
**Решение:** Программа автоматически настраивает кодировку UTF-8. Если проблема сохраняется, попробуйте:
```bash
chcp 65001
python main.py
```

### Проблема: Таймаут при запросе к API
**Решение:** Проверьте интернет-соединение. Программа ждет ответа 10 секунд.

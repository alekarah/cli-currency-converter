# CLI-утилита - Конвертер валют

Конвертер валют с загрузкой актуальных курсов через API. Две независимые реализации на Go и Python.

## 🎯 Описание

Простая и удобная CLI-утилита для конвертации валют с получением актуальных курсов через бесплатный API [exchangerate-api.com](https://www.exchangerate-api.com/).

### ✨ Возможности

- 🌐 Получение актуальных курсов валют через API
- 💱 Конвертация между любыми валютами (более 150)
- 🎨 Цветной вывод в консоли
- ⚡ Два режима работы: интерактивный и с аргументами командной строки
- 📊 Отображение курса и времени, прошедшего с последнего обновления
- 🔤 Поддержка ввода валют в любом регистре (USD, usd, Usd)
- 📝 **История конвертаций** - автоматическое сохранение всех операций в JSON файл
- 📤 **JSON вывод** - флаг `--json` для использования в скриптах и автоматизации
- 📊 **CSV вывод** - флаг `--csv` для экспорта данных

## 💡 Примеры использования

### Базовая конвертация

Валюты можно вводить как **большими**, так и **маленькими** буквами:

```bash
# Go версия
go run main.go usd rub 100
go run main.go EUR usd 50

# Python версия
python main.py Rub eur 1000
python main.py GBP JPY 200
```

**Интерактивный режим:**
```bash
# Программа сама запросит все необходимые данные
go run main.go
# или
python main.py
```

### История конвертаций

Все конвертации автоматически сохраняются в `history.json`. Просмотр:

```bash
go run main.go --history
# или
python main.py --history
```

### Конфигурационный файл

Скопируйте `config.json.example` в `config.json` и настройте под себя:

```json
{
  "default_from": "USD",
  "default_to": "RUB",
  "output_format": "text"
}
```

### Множественная конвертация и табличный режим

Передайте несколько валют через запятую, флаг `--table` для табличного вывода:

```bash
go run main.go USD RUB,EUR,CNY 100
go run main.go --table USD RUB,EUR,CNY,GBP,JPY 100
```

### JSON и CSV вывод

Флаги `--json` и `--csv` для использования в скриптах:

```bash
go run main.go --json USD RUB 100
go run main.go --csv USD RUB 100
```
